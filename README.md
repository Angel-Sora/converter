## ✨ Возможности

- 🖼️ **Конвертация изображений** - JPG, PNG, BMP, WEBP, PDF
- 📄 **Конвертация документов** - TXT, DOCX, HTML, MD, PDF
- 📊 **Конвертация таблиц** - CSV, Excel, JSON, HTML, SQLite
- 📦 **Конвертация данных** - JSON, YAML, XML, CSV
- 📁 **Работа с архивами** - Распаковка ZIP, TAR, GZ
- 🎨 **Красивый GUI интерфейс** - Интуитивно понятный и простой
- ⚡ **Пакетная обработка** - Конвертация множества файлов за раз
- 🧵 **Многопоточность** - Ускорение обработки больших объемов данных
- 🔍 **Автоопределение кодировки** - Корректная работа с разными языками
- 📊 **Детальная статистика** - Отчеты о конвертации с компрессией
- 🗂️ **Рекурсивный обход папок** - Обработка всех вложенных папок
- 🔄 **История операций** - Логирование всех действий
- 📈 **Прогресс-бар** - Визуальный индикатор выполнения
